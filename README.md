# Priority-Based Hospital Management System Using Heap

## Overview

This project implements a Priority-Based Hospital Management System using the Heap data structure. The system manages patient records and ensures that patients with higher priority (critical conditions) are treated before patients with lower priority.

The project demonstrates the practical application of priority queues and heap operations in healthcare management systems.

## Features

* Patient registration
* Priority-based patient scheduling
* Automatic treatment order generation
* Insert new patients into the queue
* Remove treated patients from the queue
* Display waiting patient list
* Efficient priority management using Heap

## Problem Statement

In hospitals, patients often require different levels of urgency for treatment. A first-come-first-served approach may delay treatment for critical patients. This system uses a Heap-based Priority Queue to ensure that patients with the highest priority receive medical attention first.

## Technologies Used

* C Programming Language
* Heap Data Structure
* Priority Queue
* Arrays and Functions

## How It Works

1. A patient is registered with a priority level.
2. The patient is inserted into the Heap.
3. The Heap automatically maintains priority order.
4. The patient with the highest priority is served first.
5. After treatment, the patient is removed from the queue.
6. The process continues until all patients are served.
